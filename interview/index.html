    # 认识考试

    交流感（文化、做事），考做人做事，如何看待事情，理性看待各类社会现象，自身知识储备丰富，看问题的视野格局大，把握住重点，处理好各种人际关系、完成好各类任务

    # 1.面试考什么（主流题型1、3、4、5）

    # 综合分析题：（重要）

    ​ 社会现象——对社会热点怎么看？

    ​ 政策行为——党or国家or高校or事业单位出台一个规定，谈一谈（eg.对限塑令怎么看？）

    ​ 名言警句——对xxx一句话谈谈你的理解（谈谈对习说的”我将无我，不负人民“这句话）

    # 自我认知题：（几乎不考了，不能透露个人信息）

    ​ 自我介绍、谈谈优势和劣势

    # 人际关系题：（基础，较为重要，主要是情景模拟）

    ​ 认识领导、同事、自己、群众分别的定位是什么？怎么处理人际与工作？

    ​ 成熟的心态

    # 组织管理题：（做事——追求效果）

    ​ 调研、宣传、活动、会议等

    ​ 流程and做法

    ​ 做事需要创意，不能太套路，处理好细节

    # 应急应变题：（积累——来源于实际工作）不确定性，无规律

    ​ 走路上，前面电线杆倾斜了，快倒塌了你怎么办

    ​ 你是村干部，村里下冰雹把村民砸伤了怎么办

    ​ 开会ppt放不出来

    # 实务处理题（河南的多）

    ​ 主要考基层实务如何处理

    # 思维：

    ​ 一个漫画题，如果是反映社会现象——>社会现象题，如果是告诉道理——>名言警句题or态度观点题

    ​ 演讲题——>态度观点题

    # 2.认识考官

    ​ 七位考官(可能考官凶、不理你)

    ​ 流畅度、精神面貌、自信、微表情、眼神

    ​ 基本上一人15min

    # 3.认识对手

    形式上、内容上、流畅度、个性化

    四天模板课

    专业、性格、经历、储备

    发挥、状态

    平常心

    # 4.认识考场

    ​ 7个考官，去掉最高分&最低分，取平均分

    ​ 另外计时人员等

    ​ 建立自信

    # 5.认识规则

    不能迟到！

    每个签都是好签，笑死

    1-3号是样本，考官谨慎，更仔细，最后的签需简洁

    候考室（有人搞心态，笑死）——>回顾&有耐心

    考试：敲门（点两下）——>进门，站座位边，问好（各位考官，大家上午好，我是今天的xx号考生）

    ——>考官会说请坐——>坐下，手放桌上，看考官，先别看题本——>考官读导语&考试规则——>可思考一道题回答一道题也可一次性思考全部题再全部作答，无固定说法——>结束后，站起来，谢谢各位考官&鞠躬，离开

    不打领带

    看天气和空调是否外套

    3-5cm鞋

    ​ 人民时评——人民网、人民日报

    # 人际关系

    # 面试中的同事

    ​ 1.着眼合作多理解

    ​ 2.立足长远多承担

    ​ 3.分工协作多沟通

    ​ 4.相互学习促成长

    # 面试中的领导（你是个新人）

    ​ 1.尊重领导权威

    ​ 2.适应领导风格

    ​ 3.服从领导安排

    ​ 4.请示领导意见

    ​ 5.完成领导任务

    ​ 6.汇报工作结果

    # 面试中的下属（你是个组长or负责人，比较少）

    ​ 1.理解下属有苦衷

    ​ 2.主动解其难和蒙

    ​ 3.用好长处多鼓励

    ​ 4.以身作则常带动

    # 面试中的群众（亲友找你帮忙）

    ​ 1.放低姿态笑脸迎

    ​ 2.人民至上记心中

    ​ 3.情理利法解问题

    ​ 4.兼顾灵活守原则

    # 答题思路

    ​ 对于xxx事情，你该怎么办？

    ​ 看法：态度、认识、原因、影响、原则等

    ​ 认识：领导之所以给我安排很多工作是因为当前单位事情比较多，人手不足，我应该迎难而上

    ​ 办法：办人办事（现在与将来）

    # 经典题目：

    1、你和小王互换轮岗，你来到新的部门后，领导经常叫小王来处理本属于你工作职责的事，而只给你安排一些琐事，你怎么办?

    ​

    ​ 互换轮岗是一个好事情，本质上是一个工作中常见的工作方式，它有利于增强我们的综合素质、提高工作能力、拓宽我们对业务了解的面，更为重要的是，和同事之间交换岗位，能够了解彼此的工作岗位，在今后的工作当中合作起来会更加的方便和顺畅，合作的分歧会更加的少，因为我认为这是一个非常好的事情。

    新的部门——>面临新的人际关系、新的工作要求、新的工作流程、新的工作环境、具备新的工作能力

    （真话不全说，假话全不说）

    ​ 我是新人——>可能领导把这些任务交给我后无法绝对的放心，而小王有处理这方面事情的经验，因而交给他更加稳当

    ​

    ​ 各位考官，考生开始答题。到达单位之后，单位里安排我和小王进行一个互换轮岗，这在我们公职单位是一个比较好的工作方式，本质上它就是一种培训的方式。互换轮岗有利于增强我们的综合素质、提高工作能力、拓宽我们对业务了解的面。另外，和同事之间互换轮岗，能够了解彼此的工作岗位，也有利于在今后的工作中合作得更加方便和顺畅，所以我觉得互换轮岗这个事情非常的好。那么互换轮岗就决定了我所到达的是新的部门，而新的部门就决定了我面临的是新的领导、新的同事、新的工作环境，需要我有一定的时间成长和适应，在这种情况下，在短时间内，我无法成长的很快，做一些关键性工作可能无法保质保量的将其完成，因此，领导会认为在一些关键工作上让小王去做会更加有把握，进而做了这样的安排。从我的态度上应该去理解，但从行为上，我应该要知道工作职责内的事情还是应该我去做，我应该让自己成长起来，因此我会这么办：

    ​ 首先，我会将领导交给我的琐事事无巨细的完成，尤其是一些看似简单的事情，比如发通知、打印材料。我会以一个高标准来完成这些事情，在做琐事的过程中增加领导对我的了解，同时也逐步的去熟悉单位的同事和工作，因此我会提前把这些事情做好。

    ​ 第二，在职责内的事情，我会在尽快做好琐事的基础之上，有更多的时间去协作好小王，在小王处理我职责上的事情时，我可以去帮助和协助他，在这个过程中向他学习，能够尽快的掌握好工作职责内的事情，当然在这个过程中，考虑到小王现在是在我原来的岗位上，我也可以去多分享我之前岗位上一些重点工作的经验。这样或许对他也有一定的帮助。

    ​ 第三，在新的部门，我们的工作情况也应该及时的和领导汇报，遇到问题也应该及时的向领导进行请示，那么我相信，随着我能力的不断提升，领导对我的不断了解，我去做我职责内的事情自然是一个自然而然地事情，我也会在这个新的部门把工作去完成好，让自己也成长得越来越好。考生作答完毕。
